Простая библиотека для базового анализа текста.

## Возможности
Подсчет слов и символов
Анализ тональности текста (базовый)
Извлечение ключевых слов

## Установка
Скопируйте файл себе в пайтон

## Использование
```python
from text_analyzer import TextAnalyzer

analyzer = TextAnalyzer("Пример текста для анализа.")
print(analyzer.word_count())
print(analyzer.character_count())
